# Backend

## Response format
| Key   	| Description                                                        	|                    	|
|-------	|--------------------------------------------------------------------	|--------------------	|
| error 	| If there's no error, returns false, if not returns the following   	|                    	|
|       	| code                                                               	| The error code     	|
|       	| Message                                                            	| Message to display 	|
| data  	| The data returned from the call. If there's an error returns false 	|                    	|
